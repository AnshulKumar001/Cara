# 🛒 Cara – Ecommerce Website

This repository contains **Cara**, a frontend **Ecommerce website** created for **learning and practice**.
The website is deployed using **GitHub Pages**.

🌐 Live Demo: [https://anshulkumar001.github.io/Cara/](https://anshulkumar001.github.io/Cara/)

## 🔹 About Cara

Cara is a **static Ecommerce website** that demonstrates the basic structure and design of an online shopping platform.
It includes product sections, layout design, and user-friendly UI.
